# print number in fixed point decimal notation, but no trailing zeros

print number in fixed point decimal notation, but no trailing zeros

## Usage

``` r
formatf_without_trailing_zeros(nr)
```

## Arguments

- nr:

  The input number.

## Value

The input number, as a string in fixed point decimal notation, with
trailing zeros removed.

examples formatf_without_trailing_zeros(37.5600)
